# MonoClock [![CircleCI](https://circleci.com/gh/srijs/node-monoclock.svg?style=svg)](https://circleci.com/gh/srijs/node-monoclock)

> Monotonic Clock

## Installation

```
npm install monoclock
```

## Usage

```
import * as monoclock from 'monoclock';

monoclock.getTime();
// { sec: 20243, nsec: 723136835 }
```
